## Rodando o projeto

1. [Instalar o node](https://nodejs.org/en/download);
2. Após instalação, verificar se o npm está funcionando (rodar `npm --version` num novo terminal);
3. Acessar pasta do projeto via terminal;
4. Instalar globalmente o yarn (`npm i -g yarn`);
5. Executar comando `yarn`;
6. Executar comando `yarn vite`;
7. Acessar [http://localhost:5173/](http://localhost:5173/)
